## ğŸ” Step 2: Inspect the Container

Use `docker inspect` to view complete details about the container.

```bash
docker inspect mynginx
```
<img width="871" height="252" alt="image" src="https://github.com/user-attachments/assets/d0252a1f-8cf1-4f2b-ac13-a8d985181194" />


This command outputs detailed JSON information, including:

* Container ID
* Image name and version
* Mounted volumes
* Network settings (IP address, ports)
* Environment variables
* Start time and status

Example (shortened):

```json
[
  {
    "Id": "a1b2c3d4e5f6...",
    "Name": "/mynginx",
    "State": {
      "Status": "running",
      "StartedAt": "2025-11-03T10:15:30Z"
    },
    "NetworkSettings": {
      "IPAddress": "172.17.0.2"
    }
  }
]
```
